Differences Between HTML and HTML5

HTML HTML5
HTML stands for Hypertext Markup Language. HTML5 stands for Hypertext Markup Language 5.
HTML4 was the previous version of HTML. HTML5 is the latest version of HTML.
HTML4 had limited support for multimedia elements. HTML5 provides native support for audio and video elements, eliminating the need for third-party plugins like Flash.
HTML4 didn't have the <canvas> element. HTML5 introduced the <canvas> element, which allows for dynamic 2D graphics rendering directly in the browser.
